1. git 설치

2. git config --global user.name/user.email ### >> 유저의 이름과 이메일 등록

3. git으로 관리할 프로젝트 폴더로 이동해서 터미널 열기(or git bash)

4. 파일 생성 후 저장

5. git inhit 명령어로 git 저장소 생성

6. git add * 로 감시할 파일 추가

7. git commit "#" 으로 저장소에 저장하기 (파일 또는 스냅샷)

8. git status로 저장할 수 있는 상태인지 검사

9. git log --oneline --decorate --graph --all 로 커밋 이력 확인하기

10. git commit -a  ===  git add *

11. git commit -m "설명" :: 

12. git diff 로 파일 상태 비교(스냅샷이 찍힐 내역 확인)

13. github 가입 후 레파지토리 생성

14. 생성할 레파지토리 주소를 등록 git remote add 리모트저장소이름(origin) 주소(http or ssh)

15. git push 저장소 이름(origin) 브랜치이름(master) : 리모트 저장소에 작업중인 가지의 이력 등록

16. git clone 주소 : 리모트 저장소를 최초로 받아오는 방법

17. git fetch 주소 or git pull 주소 : 다른 사람의 작업 이력을 나의 이력과 통합하는 방법
